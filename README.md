## How to run

```bash
$ git clone https://github.com/Bullseye-0xD4763/Computing-Laboratories-I
$ cd BlockDude/src
$ ghc -main-is Main -outputdir Tarefa5 -o BlockDude Tarefa5_2021li1g012
$ ./BlockDude
```

## Dependencies

- GHC (Glasgow Haskell Compiler)
- gloss
- gloss-juicy
- HUnit (optional)
