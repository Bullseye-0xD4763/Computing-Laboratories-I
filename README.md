## How to run

```bash
$ git clone https://github.com/DigoqueDigo/BlockDude.git
$ cd BlockDude/src
$ ghc -main-is Main -outputdir Tarefa5 -o BlockDude Tarefa5_2021li1g082
$ ./BlockDude
```

## Dependencies

- GHC (Glasgow Haskell Compiler)
- gloss
- gloss-juicy
- HUnit (optional)
